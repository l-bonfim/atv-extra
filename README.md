# Atividade extra PIF

> Este programa em C implementa um sistema simples de gerenciamento de clientes bancários utilizando arquivos binários. Os dados de cada cliente (nome, número da conta e saldo) são armazenados em uma estrutura (`struct`) e gravados no arquivo `data.dat`.
>
> O sistema permite cadastrar clientes em posições específicas do arquivo, consultar clientes pelo número da conta, atualizar saldos, remover contas e listar todos os registros armazenados. Para a exclusão, é criado um arquivo temporário contendo apenas os registros que devem ser mantidos, substituindo posteriormente o arquivo original.
>
> Além disso, o programa demonstra o uso de funções importantes de manipulação de arquivos em C, como `fopen()`, `fread()`, `fwrite()`, `fseek()`, `rewind()`, `remove()` e `rename()`, servindo como um exemplo prático de persistência de dados em arquivos binários.
